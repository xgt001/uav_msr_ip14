uav_msr_ip14
============

Image Processing Module for UAV MSRIT Mission for 2014

Components
---------------------

* Automatic Blob Extractor based on Python OpenCV
* Manual Blolb Cropper using PyQT
* A Custom GUI for Gphoto2 with local and remote execution
